## Environment

1. PHP 8.1
2. MySQL
3. Codeigniter 3.1.13
4. Bootstrap 4.6
5. Jquery 3.5.1
6. Library DataTable 1.12.1

## Installation

1. download or clone the project
2. import database from _employee.sql_
3. run the project from your localhost or etc
